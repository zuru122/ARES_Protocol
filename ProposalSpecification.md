## Ares Protocol Specification

### Overview

Ares is a treasury execution system built accross four (4) independent modues, which handles all funds movement, hence, all funds most pass all this fourstages: which includes:

- proposal
- authorization
- time delay
- execution

#### Proposal Creation

In Proposal Creation, anyone can propose treaury action by calling `creatProposal()`on the Proposal managemnt `ProposalMg`. they must have 0.1 ETh as a deposit to prevent spaming.

During creation, a unique ID is generated by hashing all proposal fields together. he proposal cannot move forward for at least 1 hour (commit phase).
